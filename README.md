# graphql-access-control-example

## Requirements

* Terraform
* AWS account
* AWS CLI configured

## Install

* ```terraform init```
* ```terraform apply```

## Use

* Go to the AppSync console
* Log in with users (```user1```, ```user2```, ```user3```, ```admin1```, ```admin2```); all passwords are ```Password.1```

## Cleanup

* ```terraform destroy```
